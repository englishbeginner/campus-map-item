# 前端搭建与前后端联调完全指南

## 第一章：项目对接总览

### 1.1 前端技术栈建议
- **框架**：Vue 3.4
- **构建工具**：Vite 5
- **UI组件库**：Vant 4
- **状态管理**：Pinia
- **地图SDK**：高德地图JS API v2.0
- **HTTP客户端**：Axios

### 1.2 前后端交互数据流图
前端事件 → 调用接口 → 后端处理 → 返回数据 → 前端渲染

### 1.3 核心对接约定
- **坐标系统一**：GCJ-02（火星坐标）
- **请求/响应Content-Type**：`application/json`
- **鉴权方式**：Header中携带 `Authorization: Bearer <token>`
- **响应数据结构**：统一使用 `Result<T>` 格式，包含 `code`、`msg`、`data` 字段

## 第二章：前端项目初始化与核心配置

### 2.1 从零开始的初始化命令
```bash

# 进入项目目录

# 安装核心依赖
npm install vant@4 pinia axios @amap/amap-jsapi-loader

```

### 2.2 环境变量配置
创建 `.env.development` 文件：
```env
# 后端API基础地址
VITE_API_BASE_URL=http://localhost:8080

# 高德地图API Key
VITE_AMAP_KEY=your_amap_api_key
```

### 2.3 Vite配置文件
创建或修改 `vite.config.js` 文件：
```javascript
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'

// https://vitejs.dev/config/
export default defineConfig({
  plugins: [vue()],
  server: {
    proxy: {
      '/api': {
        target: 'http://localhost:8080',
        changeOrigin: true,
        rewrite: (path) => path
      }
    }
  }
})
```

### 2.4 Axios封装
创建 `src/utils/request.js` 文件：
```javascript
import axios from 'axios'

// 创建axios实例
const request = axios.create({
  baseURL: import.meta.env.VITE_API_BASE_URL,
  timeout: 10000
})

// 请求拦截器
request.interceptors.request.use(
  config => {
    // 从localStorage获取token
    const token = localStorage.getItem('token')
    if (token) {
      config.headers.Authorization = `Bearer ${token}`
    }
    return config
  },
  error => {
    return Promise.reject(error)
  }
)

// 响应拦截器
request.interceptors.response.use(
  response => {
    const res = response.data
    if (res.code !== 200) {
      // 处理错误
      console.error(res.msg)
      return Promise.reject(res)
    }
    return res.data
  },
  error => {
    console.error('网络错误:', error)
    return Promise.reject(error)
  }
)

export default request
```

## 第三章：后端接口对接详解

### 3.1 POI模块接口对接

#### 3.1.1 接口：根据视野范围获取POI列表
- **后端路径**：`GET /api/v1/pois/list-in-bounds`
- **前端API封装代码**：
  ```javascript
  import request from '@/utils/request'
  
  export function getPOIListInBounds(params) {
    return request({
      url: '/api/v1/pois/list-in-bounds',
      method: 'get',
      params
    })
  }
  ```
- **入参说明**：`params` 需包含 `minLng`, `maxLng`, `minLat`, `maxLat`
- **出参数据结构说明**：
  ```json
  [
    {
      "id": 1,
      "name": "第一教学楼",
      "alias": "一教",
      "type": 1,
      "centerLng": 116.327156,
      "centerLat": 39.984706,
      "boundary": "[[116.326156, 39.984206], [116.328156, 39.984206], [116.328156, 39.985206], [116.326156, 39.985206]]",
      "photoUrl": "https://example.com/photo1.jpg",
      "description": "学校主要教学楼之一",
      "status": 1,
      "createTime": "2026-03-25T10:00:00",
      "updateTime": "2026-03-25T10:00:00"
    }
  ]
  ```
- **前端调用时机**：当地图 `moveend` 或 `zoomend` 事件触发时

#### 3.1.2 接口：关键词搜索POI
- **后端路径**：`GET /api/v1/pois/search`
- **前端API封装代码**：
  ```javascript
  import request from '@/utils/request'
  
  export function searchPOI(keyword) {
    return request({
      url: '/api/v1/pois/search',
      method: 'get',
      params: { keyword }
    })
  }
  ```
- **入参说明**：`keyword` 为搜索关键词
- **出参数据结构说明**：与 `list-in-bounds` 接口相同
- **前端调用时机**：用户在搜索框输入关键词并提交时

#### 3.1.3 接口：获取POI详情
- **后端路径**：`GET /api/v1/pois/{id}`
- **前端API封装代码**：
  ```javascript
  import request from '@/utils/request'
  
  export function getPOIDetail(id) {
    return request({
      url: `/api/v1/pois/${id}`,
      method: 'get'
    })
  }
  ```
- **入参说明**：`id` 为POI的唯一标识
- **出参数据结构说明**：与 `list-in-bounds` 接口返回的单个对象相同
- **前端调用时机**：用户点击地图上的POI标记时

## 第五章：常见问题排查（FAQ）

### 5.1 跨域问题怎么办？
- **原因**：前端和后端运行在不同的端口，浏览器会阻止跨域请求
- **解决方案**：
  1. 后端已配置CORS，允许所有来源的请求
  2. 前端使用Vite代理，将 `/api` 请求代理到后端的 `http://localhost:8080`
  3. 检查 `vite.config.js` 中的代理配置是否正确

### 5.2 日期格式/BigDecimal精度在前端如何处理？
- **日期格式**：后端返回的日期时间格式为ISO 8601（如 `2026-03-25T10:00:00`），前端可以使用 `new Date()` 进行转换
- **BigDecimal精度**：后端返回的经纬度等数值类型为Double，前端会自动处理为JavaScript的Number类型，精度足够使用

### 5.3 如何将后端返回的JSON轮廓坐标传给高德地图API绘制多边形？
- **步骤**：
  1. 后端返回的 `boundary` 字段为JSON字符串，需要先解析为数组
  2. 使用高德地图API的 `AMap.Polygon` 类绘制多边形
  3. 示例代码：
    ```javascript
    // 解析边界坐标
    const boundary = JSON.parse(poi.boundary)
    
    // 创建多边形
    new AMap.Polygon({
      path: boundary,
      strokeColor: '#0091ff',
      strokeOpacity: 0.6,
      strokeWeight: 2,
      fillColor: '#0091ff',
      fillOpacity: 0.35
    }).setMap(map)
    ```

### 5.4 如何处理地图的缩放级别和视野变化事件？
- **监听事件**：
  ```javascript
  // 监听地图缩放结束事件
  map.on('zoomend', () => {
    // 获取当前视野范围
    const bounds = map.getBounds()
    const southWest = bounds.getSouthWest()
    const northEast = bounds.getNorthEast()
    
    // 调用POI列表接口
    getPOIListInBounds({
      minLng: southWest.getLng(),
      maxLng: northEast.getLng(),
      minLat: southWest.getLat(),
      maxLat: northEast.getLat()
    })
  })
  
  // 监听地图移动结束事件
  map.on('moveend', () => {
    // 同样获取视野范围并调用接口
  })
  ```

### 5.5 如何实现地图的飞点（FlyTo）效果？
- **使用高德地图API**：
  ```javascript
  // 飞点到指定坐标
  map.setCenter([poi.centerLng, poi.centerLat])
  map.setZoom(18)
  
  // 或使用动画效果
  map.panTo([poi.centerLng, poi.centerLat], {
    duration: 1000
  })
  ```
